# WorkSphere - Gestion visuelle du personnel

## Contexte du projet
WorkSphere est une application web dédiée à la gestion visuelle et interactive du personnel au sein des locaux de l’entreprise. 
Elle permet d’organiser, déplacer et assigner les employés sur un plan d’étage en respectant les contraintes liées aux rôles et zones autorisées.

## Fonctionnalités
- Ajouter, déplacer et supprimer des employés.
- Gestion des rôles et restrictions par zone.
- Interface responsive pour ordinateurs, tablettes et mobiles.
- Validation des formulaires avec `required` et Regex.
- Modale d’ajout avec prévisualisation de photo.
- Formulaire dynamique pour ajouter plusieurs expériences professionnelles.
- Zones vides obligatoires affichées en rouge.
- Limitation du nombre d’employés par zone.
- (Bonus) Drag & Drop et réorganisation automatique.
- Sauvegarde de l’état du plan dans `localStorage`.

## Technologies utilisées
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (ES6)
- Git/GitHub pour versionnement
